

# План за калкулатор, аналогов часовник и преглед на разходите

## 1. Разходи — вече съществува ✅
Финансовият модул (`/finance`) вече има пълна система за разходи:
- Добавяне на разходи с категория, сума, дата, описание
- Таблица с всички разходи и възможност за триене
- 7 категории (Доставка, Маркетинг, Заплати, Наем, Консумативи, Връщания, Други)
- Обобщение с печалба = платени + частични - разходи
- Фактури с пълна CRUD функционалност

**Не е нужна промяна** — модулът е завършен.

## 2. Плаващ калкулатор с история

Нов компонент `FloatingCalculator` — бутон който отваря draggable панел:

- Стандартен калкулатор (цифри, +, -, ×, ÷, %, =, C)
- Може да се закачи вляво/вдясно чрез drag
- **История по дни** — записва всяко изчисление в `localStorage` с timestamp
- Бутон за изтриване на цялата история
- Интегриран в `Index.tsx` (поръчки) и `Inventory.tsx` (склад) като floating бутон в header-а

### Файлове
| Файл | Действие |
|------|----------|
| `src/components/FloatingCalculator.tsx` | **НОВ** — калкулатор компонент с drag, история |
| `src/pages/Index.tsx` | Добавяне на `<FloatingCalculator />` |
| `src/pages/Inventory.tsx` | Добавяне на `<FloatingCalculator />` |

## 3. Аналогов часовник (Bulgaria timezone)

Нов компонент `AnalogClock` — красив SVG аналогов часовник:

- SVG базиран с стрелки (часова, минутна, секундна)
- Римски или арабски цифри, елегантен дизайн с тъмен/светъл режим
- Показва точно българско време (`Europe/Sofia` timezone)
- Позициониран в header-а на двете страници (Index + Inventory) като малък 40px часовник
- Tooltip с точния час при hover

### Файлове
| Файл | Действие |
|------|----------|
| `src/components/AnalogClock.tsx` | **НОВ** — SVG аналогов часовник |
| `src/pages/Index.tsx` | Добавяне в header |
| `src/pages/Inventory.tsx` | Добавяне в header |
